# Privacy Policy

**Last Updated:** February 14, 2026

## Overview

Summarizer is a Chrome extension designed to summarize text on websites using artificial intelligence. We value your privacy and this policy explains how we handle your personal data.

## Data Collection

### User Data
- **No personal data collected:** We do not collect your name, email, IP address, or identification information.
- **Summarized text:** Text that is summarized is not stored on our servers.
- **Temporary processing:** Text is only temporarily processed during summarization and immediately deleted.

### API Keys
- **User API keys:** When you use your own API key, it is encrypted and stored locally only in your Chrome storage.
- **No server submission:** Your API keys are not sent to or stored on our servers.

### Telemetry Data
- **Anonymous usage statistics:** We collect only anonymous usage statistics (installation count, active user count).
- **No identification data:** This data contains no personal identification information.

## Security

- **Encryption:** Locally stored API keys are encrypted with AES-GCM
- **HTTPS:** All network communication is done over HTTPS
- **Secure storage:** Chrome's secure storage features are used
